# Setup

Windows:

Install 5.5.0 release.

Ubuntu:

```bash
sudo apt-get update
sudo apt-get install tesseract-ocrsudo apt-get update
sudo apt-get install tesseract-ocr
```

Check installed and has version.

```bash
tesseract --version
```
